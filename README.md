# Meu Primeiro Site – Sistema de Login

Este projeto foi desenvolvido como parte do meu aprendizado em programação,
com o objetivo de praticar os fundamentos do desenvolvimento web utilizando
HTML, CSS e JavaScript puro.

## Funcionalidades
- Cadastro de usuário
- Login com validação de senha
- Controle de tentativas de acesso
- Exibição de mensagens de erro e sucesso
- Acesso protegido a uma página de painel
- Logout do sistema
- Armazenamento de dados no localStorage
- Envio do formulário via botão ou tecla Enter

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage

## Objetivo do Projeto
Aprender os fundamentos do desenvolvimento web, incluindo:
- Estruturação de páginas com HTML
- Estilização com CSS
- Manipulação do DOM com JavaScript
- Eventos e formulários
- Organização de arquivos
- Controle de fluxo e validações básicas

## Observações
Este projeto não possui backend e não deve ser utilizado em produção.
Foi desenvolvido exclusivamente para fins educacionais.
