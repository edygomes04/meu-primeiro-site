# Meu Primeiro Site (ESLEY)

Este é meu primeiro projeto web, desenvolvido como parte do meu aprendizado em programação. Meu "ponta pé" inicial. Estou animado!

## Funcionalidades

- Página web simples
- Campo de senha
- Validação de acesso com JavaScript
- Mensagens de erro e sucesso
- Envio do formulário pelo botão ou tecla Enter

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do projeto

meu-primeiro-site/
├── index.html
├── css/
│   └── style.css
└── js/
    └── script.js

## Objetivo

Aprender os fundamentos de desenvolvimento web, incluindo estrutura, estilo e lógica com JavaScript.
